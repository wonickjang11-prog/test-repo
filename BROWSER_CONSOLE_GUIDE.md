# 브라우저 콘솔에서 스크립트 실행하는 방법 (초보자용)

## 📖 단계별 가이드

### 1단계: Gemini Gem 페이지 열기

1. 웹 브라우저(Chrome, Edge, Brave 등)를 엽니다
2. 다음 URL을 주소창에 붙여넣습니다:
   ```
   https://gemini.google.com/gem/2978c017455c/f84d19cbb2532ab8
   ```
3. **Enter** 키를 누릅니다
4. 필요하면 Google 계정으로 로그인합니다
5. 페이지가 완전히 로드될 때까지 기다립니다

---

### 2단계: 개발자 도구 열기

다음 방법 중 하나를 선택하세요:

#### 방법 A: 키보드 단축키 (가장 빠름)
- **Windows/Linux**: `F12` 키를 누릅니다
- **Mac**: `Cmd + Option + I` (또는 `Cmd + Opt + I`)

#### 방법 B: 마우스 메뉴
1. 페이지의 빈 공간에서 **마우스 오른쪽 클릭**
2. 메뉴에서 **"검사"** 또는 **"Inspect"** 클릭

#### 방법 C: 브라우저 메뉴
- **Chrome**:
  1. 우측 상단 점 3개 (⋮) 클릭
  2. **도구 더보기** → **개발자 도구**

- **Edge**:
  1. 우측 상단 점 3개 (⋯) 클릭
  2. **도구 더보기** → **개발자 도구**

**결과**: 브라우저 하단이나 옆에 개발자 도구 패널이 열립니다

---

### 3단계: Console 탭으로 이동

1. 개발자 도구 패널 상단에 여러 탭이 보입니다:
   ```
   Elements | Console | Sources | Network | ...
   ```

2. **Console** 탭을 클릭합니다

**결과**:
- 텍스트를 입력할 수 있는 큰 영역이 나타납니다
- `>` 기호 옆에 커서가 깜빡입니다
- 여기에 JavaScript 코드를 입력할 수 있습니다

---

### 4단계: 스크립트 코드 복사하기

1. `gemini_export_browser.js` 파일을 엽니다
   - 텍스트 에디터로 열거나
   - GitHub에서 파일을 열어서

2. **전체 내용을 복사**합니다:
   - **Windows/Linux**: `Ctrl + A` (전체 선택) → `Ctrl + C` (복사)
   - **Mac**: `Cmd + A` → `Cmd + C`

**복사할 코드 예시:**
```javascript
(function() {
    // 모든 텍스트 콘텐츠 추출
    const extractContent = () => {
        ...
    };
    ...
})();
```

---

### 5단계: 콘솔에 코드 붙여넣기

1. Console 탭의 `>` 기호 옆을 **클릭**합니다

2. **붙여넣기**:
   - **Windows/Linux**: `Ctrl + V`
   - **Mac**: `Cmd + V`

3. 여러 줄의 코드가 콘솔에 나타납니다

**주의사항:**
- 일부 브라우저에서 경고 메시지가 나올 수 있습니다:
  ```
  ⚠️ Warning: Don't paste code into the console...
  ```
- 이 경고는 보안을 위한 것입니다
- 신뢰할 수 있는 코드만 붙여넣으세요
- 계속하려면 콘솔에 `allow pasting` 을 입력하라고 할 수 있습니다

---

### 6단계: 코드 실행하기

1. 코드를 붙여넣은 후 **Enter** 키를 누릅니다

2. 코드가 실행되면서:
   - 콘솔에 진행 메시지가 나타납니다
   - 페이지 콘텐츠를 분석합니다
   - 자동으로 `.txt` 파일을 다운로드합니다

**성공 메시지:**
```
✅ 다운로드 완료!
추출된 콘텐츠 길이: 12345 문자

--- 미리보기 (처음 500자) ---
...
```

---

### 7단계: 다운로드된 파일 확인

1. 브라우저 하단에 다운로드 알림이 나타납니다

2. 파일 이름 예시:
   ```
   gemini_research_2026-01-23.txt
   ```

3. 다운로드 폴더를 확인합니다:
   - **Windows**: `C:\Users\사용자명\Downloads\`
   - **Mac**: `~/Downloads/`
   - **Linux**: `~/Downloads/`

4. 파일을 열어서 내용을 확인합니다

```

```


## ⚠️ 주의사항

### 1. "allow pasting" 입력 요구
브라우저에서 처음 붙여넣을 때 다음과 같은 메시지가 나올 수 있습니다:

```
⚠️ Warning: Don't paste code into the DevTools Console...
Type 'allow pasting' below to allow pasting
```

**해결 방법:**
1. 콘솔에 `allow pasting` 을 **정확히** 입력합니다 (따옴표 없이)
2. Enter를 누릅니다
3. 이제 코드를 다시 붙여넣을 수 있습니다

### 2. 여러 줄 코드 붙여넣기
- `Shift + Enter`를 누르면 실행하지 않고 다음 줄로 이동합니다
- 코드가 여러 줄이면 한 번에 붙여넣고 마지막에 **Enter** 한 번만 누르세요

### 3. 오류 발생 시
빨간색 오류 메시지가 나타나면:
```
❌ Uncaught TypeError: ...
```

**원인:**
- 페이지가 완전히 로드되지 않음
- Gemini 페이지 구조가 변경됨
- 로그인이 필요함

**해결:**
1. 페이지를 새로고침 (`F5`)
2. 완전히 로드될 때까지 기다림
3. 다시 시도

## 💡 추가 팁

### 코드 재사용
- 한 번 붙여넣은 코드는 위 화살표 키(↑)로 다시 불러올 수 있습니다
- 여러 Gem을 연속으로 추출할 때 유용합니다

### 콘솔 지우기
- 콘솔이 너무 복잡해지면 `clear()` 입력 후 Enter
- 또는 🚫 아이콘 클릭

### 개발자 도구 닫기
- `F12` 다시 누르기
- 또는 우측 상단 X 버튼 클릭

## 🆘 문제 해결

| 문제 | 해결 방법 |
|------|----------|
| Console 탭이 안 보임 | `>>` 버튼 클릭하여 숨겨진 탭 확인 |
| 코드 붙여넣기 안 됨 | `allow pasting` 입력 |
| 다운로드 안 됨 | 브라우저 다운로드 설정 확인 |
| 파일이 비어있음 | 페이지 완전 로드 후 재시도 |
| 오류 메시지 | 페이지 새로고침 후 재시도 |
